In Safari you can _<kbd>⌘</kbd> + <kbd>z</kbd>_ to reopen the last closed tab, but it only works once. With _retab_ you can recover as many tabs as you want.

# Install

Download it [**here**][bundle], double click the bundle and you will be reopening closed tabs in no time.

You can also check out the source code _< 40 LOC_.

# Features

+ Chrome-style Key Bindings
+ Remembers Closed Tab Position
+ Background (Faster) / Foreground Tab Opening
+ Respect _Private Browsing_ Mode
+ Unlimited Tabs
+ _Restore Tab Browsing History_ [__WIP__]

## Key Bindings

_retab_ uses Chrome-_style_ _<kbd>⌘</kbd> + <kbd>⇧</kbd> + <kbd>T</kbd>_ bindings by default, but you can change this via Settings.
